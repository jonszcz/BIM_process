Design Options

Example

A notch in the exterior wall will require the following

  • The exterior wall needs to be modified in the Shell model
  • The slab edge needs to be adjusted in the Structure model
  • The interior partitions need to be adjusted in the Core model.

Solution

All three of these models should have a design option set called Exterior Notch. The primary options should be No Notch and one Design Option could be 2' Deep Notch.

If we had a spreadsheet to outline the Design Options and attempted to unify and track the naming and scope of the design option studies would that be used?
